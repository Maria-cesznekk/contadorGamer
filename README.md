#🎮 Contador Gamer

Projeto desenvolvido em React Native com Expo

---

#Preview do APP

Oaplicativo possui:
* contador de pontos;
* botão de aumentar;
* botão de diminuir;
* botão de resetar;
* interface estilo gamer.

---

# Tecnologias utilizadas

* React Ntive
* Expo
* JvaScript

---

#Como executar o projeto

## 1. clonar o repositório

´´´
 git clone https://github.com/Maria-cesznekk/contadorGamer.git
´´´

## 2. Entrar na pasta do projeto
´´´
cd contadorGamer
´´´

## 3. Instalar dependências
´´´
npm install
´´´

## 4. Rodar o projeto
´´´
npm start
´´´
